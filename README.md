## Configuration File
config.yml

```yaml
server:
  port: 8080
logging:
  level: info
  file:
    name: app.log
    path: logs
# en/zh-CN
language: en
datasource:
  driver: mysql
  url: tcp(127.0.0.1:3306)/test?charset=utf8mb4&parseTime=True&loc=Local
  username: 
  password: 
jwt:
  secret: test
  expire: 7
verification:
  resourceEnabled: false
redis:
  addr: localhost:6379
  password:
  db: 0
```

## Swagger
### install
```shell
go install github.com/swaggo/swag/cmd/swag@latest
```
### init or update
```shell
swag init
```
### UI URL
```shell
http://localhost:8080/swagger/index.html
```
